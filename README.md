# how to start  
---
  * first clone this repo:  
  `git clone git@github.com:shyamkumaryadav/SeeDjango.git`  
  * `cd SeeDjango`  
  * `pipenv shell && pipenv install`  
  * `cd guijs && npm install && npm run build`
  * `./manage.py runserver`  
    Done !!!
